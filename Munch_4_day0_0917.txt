The five diffrent java types are,

1)int-integer, i.e., whole or counting numbers,5 6 7
2)float-number with decimals, e.g., 5.5 6.6
3)short,long,double-we will mostly use double for decimal numbers
4)char-data type is a single 16-bit Unicode character.
5)boolean-has only two possible values: true and false.
